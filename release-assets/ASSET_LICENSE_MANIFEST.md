# 素材来源与授权记录

| 类别 | 文件范围 | 来源 | 发布前动作 |
| --- | --- | --- | --- |
| 游戏角色与洞口 | `assets/resources/textures/gameplay/*.png` | OpenAI 内置图像生成工具生成，本项目定制 | 保存提示词与生成记录 |
| 游戏背景 | `gameplay_background.jpg` | OpenAI 内置图像生成工具生成，本项目定制 | 保存提示词与生成记录 |
| 图标与启动图 | `release-assets/branding/*` | OpenAI 内置图像生成工具生成并本地排版 | 运营主体确认可发布性 |
| 中文标题字形 | 启动图内栅格文字 | macOS `Heiti-SC-Medium` 本地栅格化 | 不分发字体文件；发布前确认使用条款 |
| 短音效 | `sfx_*.wav` | `tools/generate_sfx.mjs` 程序生成 | 保留生成脚本 |
| 背景音乐 | `bgm_meadow_loop.mp3` | `tools/generate_bgm.mjs` 程序生成 | 保留生成脚本与 MP3 转码参数 |
| Cocos Creator | 引擎与构建产物 | Cocos Creator 3.8.8 | 按引擎许可和平台要求发布 |

项目当前未引入第三方图库、音乐库或网络下载素材。若后续新增素材，必须记录来源网址、作者、许可类型、购买凭证及允许的发布平台。
