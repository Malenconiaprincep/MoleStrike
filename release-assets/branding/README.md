## 生成方式

- 图标：内置图像生成工具，以项目普通地鼠贴图作为角色身份参考。
- 启动图：内置图像生成工具，以最终图标作为角色、玩具锤和配色参考。
- 中文标题：使用本机 `Heiti-SC-Medium` 栅格化叠加，保证“地鼠突击队”无错字；项目不分发字体文件。
- 图像均未使用第三方品牌、平台标识或水印。

- 启动图：同一角色从洞口跃起，竖版草地场景，上方预留标题空间，无文字、无平台标识。
