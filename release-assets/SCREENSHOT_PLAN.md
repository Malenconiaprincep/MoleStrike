# 平台截图计划

必须使用开发者工具或真机实际运行画面，不使用概念图冒充游戏截图。

## 建议截图

1. 首页：标题、正式背景、开始游戏按钮和地鼠角色完整可见。
2. 普通玩法：九个洞口、分数、倒计时和普通地鼠。
3. 特殊地鼠：金色地鼠出现，并显示 `+5` 反馈。
4. 风险玩法：炸弹地鼠与红色命中反馈。
5. 高连击：显示 `Combo xN` 和“突击”难度。
6. 结算页：最终得分、最高分、评级、分享和排行榜入口。

## 验收要求

- 竖屏原始截图，不拉伸、不叠加设备边框；
- 不出现 FPS、Draw Call、vConsole 或编辑器工具栏；
- 不使用测试 AppID、调试提示、占位排行榜文案；
- 文案与实际上线版本一致；
- 至少准备一组微信真机截图和一组抖音真机截图；
- 最终数量、尺寸和格式以提交当日两个平台后台提示为准。
