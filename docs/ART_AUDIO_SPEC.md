# 美术与音效资源说明

## 美术风格

- 关键词：Q版、明亮、圆润、儿童友好、暖色调。
- 参考观感：微信小游戏、天天爱消除、羊了个羊一类轻休闲游戏。
- 主色：草地绿色、木头棕、暖黄色、浅蓝天空点缀。
- 避免：暗色、锐利边缘、高写实、恐怖表情。

- 素材由内置图像生成工具制作，提示词要求 Q 版休闲游戏风格、暖色调、无文字和无水印；角色图使用纯品红底生成后在本地移除色键。

## 资源设计说明

### 1. 背景：gameplay_background.jpg

- 内容：明亮草地场景，可带小花、浅色石子、柔和天空边缘。
- 尺寸：`750 x 1334` 或 `1536 x 2732`。
- 色彩：草地绿色为主，加入浅黄和浅蓝提高亲和力。
- 用途：主场景全屏背景。

### 2. 地鼠洞：hole_wood.png

- 内容：圆形木头洞口，外圈像木桩截面，内圈深棕色洞穴。
- 尺寸：`180 x 140`。
- 视觉重点：洞口边缘要厚，方便遮挡地鼠缩回时的底部。
- 用途：9 个洞节点复用。

### 3. 地鼠：mole_normal.png

- 内容：棕色 Q 版地鼠，大眼睛、圆脸、短手、微笑。
- 尺寸：`160 x 160`。
- 表情：可爱、搞笑，不要凶。
- 用途：普通地鼠，点击 +1 分。

### 4. 按钮

#### btn_primary.png

- 用于：开始游戏、再来一次。
- 外观：橙黄色圆角木牌或果冻按钮，边缘高光。
- 尺寸：`320 x 110`。

#### btn_secondary.png

- 用于：返回首页。
- 外观：浅木色或浅绿色，优先保持可爱风格。
- 尺寸：`280 x 96`。

#### btn_pause.png

- 用于：暂停按钮。
- 外观：圆形木质按钮，中央为暂停符号。
- 尺寸：`96 x 96`。

### 5. UI 图标

#### icon_score.png

- 内容：星星、奖牌或小锤子图标。
- 尺寸：`64 x 64`。
- 用途：分数文本左侧。

#### icon_time.png

- 内容：卡通闹钟或沙漏。
- 尺寸：`64 x 64`。
- 用途：时间文本左侧。

## 小游戏音频建议

- 开发阶段可使用 `wav`，发布阶段建议压缩为 `mp3` 或 `ogg`，微信/抖音小游戏优先 `mp3`。
- 单个音效尽量小于 100KB。
- 按钮音效和击中音效要短，避免连续点击时拖沓。
- 后续可增加独立音乐音量滑杆，目前“声音”开关统一控制音乐和音效。
- 当前背景音乐为项目内程序生成的原创循环，96 kbps 单声道，音乐音量设为 `0.22`，给命中音效保留空间。
