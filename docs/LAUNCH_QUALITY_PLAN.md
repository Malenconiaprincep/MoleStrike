# 微信/抖音小游戏上线级别路线

目标：把当前可玩 Demo 推进到“至少是游戏级别”的可发布版本。

## 1. 玩法完整度

- 已完成：60 秒打地鼠核心循环。
- 已完成：普通地鼠、金色地鼠、炸弹地鼠。
- 已完成：连击、飘字、历史最高分。
- 已完成：结算评级和新纪录提示。
- 已完成：首次开始三页新手引导、跳过功能和完成状态存档。
- 已完成：每日挑战按自然日轮换单局得分、累计命中与金色地鼠目标。
- 已完成：挑战进度本地保存，首次完成自动发放并累计勋章。

## 2. 视觉品质

- 已完成：程序绘制版 Q 版临时美术。
- 已完成：正式草地背景、普通/金色/炸弹地鼠、木质洞口资源接入。
- 已完成：资源异步预载、缓存和程序绘制降级方案。
- 已完成：正式贴图使用独立渲染子节点，避免与程序绘制组件冲突。
- 已完成：全局按钮按压回弹、分类命中光圈和炸弹震动反馈。
- 已完成：暂停遮罩淡入淡出、结算卡片登场和新纪录弹跳动画。
- 已完成：顶部状态栏和底部交互按钮根据微信/抖音安全区动态偏移。
- 已完成：所有运行时标签统一系统字体、字重和高对比文字描边。

## 3. 音频品质

- 已完成：按钮、出现、命中、倒计时结束、结算音效。
- 已完成：音效开关。
- 已完成：轻快草地主题循环背景音乐，音乐与音效独立 AudioSource 播放。
- 已完成：背景音乐压缩为 96 kbps 单声道 MP3，约 189 KB。
- 待完成：真机确认不同系统音量与静音恢复行为。

## 4. 平台适配

- 已完成：平台能力适配层，隔离分享、排行榜、震动。
- 已完成：结算分享入口。
- 已完成：平台云端最高分写入、开放数据域好友榜与主域排行榜面板。
- 已完成：`wechatgame` 项目级竖屏构建模板。
- 已完成：`bytedance-mini-game` 项目级竖屏与隐藏状态栏模板。
- 已完成：一键发布静态审计命令 `npm run verify:release`。
- 已完成：微信/抖音客户端静默登录及失败安全降级。
- 待完成：运营服务端使用正式 AppID/密钥交换登录凭证并建立用户会话。
- 待完成：真机测试，至少覆盖 iOS 和 Android。

## 5. 性能与包体

- 当前策略：正式玩法美术约 1.5 MB，音效约 244 KB，首包负载较低。
- 已完成：地鼠对象池，降低运行时节点分配。
- 已完成：全屏背景使用 JPG 压缩，透明角色使用独立 PNG。
- 已完成：启动时主动关闭 Cocos 调试统计显示。
- 已完成：自动场景仅初始化一次，管理器完成依赖注入后再激活。
- 待完成：资源压缩与分包策略。
- 待完成：低端机帧率测试。

## 6. 合规与上线材料

- 已完成：小游戏中英文名称、1024 图标和两套竖屏启动图。
- 已完成：商店短文案、完整介绍、关键词和审核备注草案。
- 已完成：隐私政策与用户协议待审模板。
- 已完成：素材来源与授权记录、真实游戏截图计划。
- 待完成：运营主体填写并审核隐私政策、用户协议。
- 待完成：按微信/抖音后台当日要求上传真实截图与资料。

## 7. 可运营与诊断

- 已完成：微信 `reportEvent`、抖音 `reportAnalytics` 的统一安全封装。
- 已完成：启动、登录、局次来源、暂停、结算指标、分享、排行榜和声音事件。
- 已完成：平台运行时错误与未处理 Promise 拒绝采集。
- 已完成：设备本地仅保留最近 20 条无身份标识诊断事件。
- 待完成：正式平台后台创建对应事件并验证数据入库。

## 8. 验收标准

- 核心玩法可连续游玩 10 局无报错。
- 暂停、重开、返回首页、结算流程稳定。
- 竖屏适配常见手机比例，无按钮遮挡。
- 音效可播放，并支持关闭。
- 微信/抖音开发者工具预览通过。
- 真机体验无明显卡顿和黑屏。
