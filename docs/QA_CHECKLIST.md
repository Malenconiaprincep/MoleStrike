# 游戏验收清单

## 编辑器预览

- 背景显示为带天空、远山和木栅栏的正式草地插画。
- 普通、金色、炸弹地鼠均显示对应正式贴图，透明边缘无品红色残留。
- 九个洞口均显示木质正式贴图；资源加载失败时仍有程序绘制画面。
- 所有按钮按下时缩放，抬起或取消触摸后恢复原始尺寸。
- 普通、金色、炸弹命中光圈颜色和强度不同。
- 命中炸弹时洞口区域轻微震动，结束后位置准确复原。
- 点击开始游戏后进入游戏界面。
- 首次点击开始时显示三页新手引导，倒计时尚未启动。
- 引导“下一步”“开始挑战”“跳过引导”均可点击且不会穿透到底层。
- 完成或跳过引导后再次开始不重复显示。
- 60 秒倒计时正常减少。
- 普通地鼠命中显示 `+1`。
- 金色地鼠命中显示 `+5`。
- 炸弹地鼠命中显示 `-3`，分数不低于 0。
- 连续命中显示 `Combo xN`。
- 漏掉地鼠后连击消失。
- 20 秒后难度显示为“加速”。
- 40 秒后难度显示为“突击”。
- 暂停按钮可以暂停和继续。
- 音效开关可以切换，重开游戏后设置仍保留。
- 背景音乐循环播放且不会被命中、按钮等短音效打断。
- 关闭“声音”后音乐和音效都停止，再次开启后音乐恢复。
- 结算页显示最终得分、历史最高、评级。
- 刷新历史最高时显示“新纪录！”。
- 结算页淡入并回弹登场，新纪录标签单独弹跳。
- 暂停遮罩淡入淡出，快速暂停/继续不会残留遮罩。
- 命中地鼠时小游戏真机有短震动。
- 结算页“分享战绩”在微信/抖音开发者工具中不报错。
- 结算页“排行榜”在未接开放数据域前安全降级，不影响继续游戏。

## 移动端适配

- iPhone 12 Pro 预览无明显裁切。
- 常见 Android 竖屏比例无按钮遮挡。
- 顶部状态 UI 不被安全区遮挡。
- 底部暂停按钮不被调试面板或安全区遮挡。
- 刘海屏顶部状态栏根据 `sys.getSafeAreaRect(false)` 自动下移。
- Home、分享、排行榜、声音和暂停等底部按钮位于安全区内。

## 发布前

- 预览和发布构建均不显示 FPS、Draw Call 等调试统计。
- 确认背景音乐 MP3 与五个短音效均进入构建包。
- 确认正式美术资源均已生成 `.meta` 并成功进入构建包。
- `npm run verify:release` 输出 0 个错误。
- 微信构建产物 `game.json` 为 `deviceOrientation: portrait`。
- 抖音构建产物为竖屏且 `showStatusBar: false`。
- 微信开发者工具预览通过。
- 抖音开发者工具预览通过。
- 分享、震动、排行榜入口均经过对应平台开发者工具测试。
- 真机连续游玩 10 局无红色报错。
